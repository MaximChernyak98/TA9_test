# Autotest for TA9 

### Setup 

1. Clone repository 
2. Create and activate virtual enviroment (Python 3.9 or 3.10)
3. Install requirements 
```bash
pip3 install -r requirements.txt
```

### Run test:

```bash
robot -d results test_cases/TA9.robot
```

### Or Run script using sh-file:

```bash
sh run_tests.sh 
```
